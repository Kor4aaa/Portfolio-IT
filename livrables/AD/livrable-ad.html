Rapport de configuration de l'authentification deux facteurs (2FA)

Contexte : La 2FA renforce la sécurité des comptes administrateurs en ajoutant une seconde vérification au-delà du mot de passe.

Solution recommandée : Microsoft Authenticator / Azure AD MFA

Politique appliquée :
– 2FA obligatoire pour tous les membres du groupe Admins du domaine
– Délai d'expiration de session : 15 minutes d'inactivité
– Verrouillage après 3 tentatives échouées (configuré via GPO)

Rapport d'installation et de configuration de Microsoft ATA

Qu'est-ce que Microsoft ATA ?
Microsoft Advanced Threat Analytics est une solution de détection des menaces sur site. Elle analyse le trafic réseau Active Directory pour détecter les comportements suspects (Pass-the-Hash, Golden Ticket, reconnaissance LDAP, etc.).

Statut : Microsoft ATA 1.9 n'est plus disponible au téléchargement. Il a été remplacé par Microsoft Defender for Identity (solution cloud).

Prérequis techniques :
– Windows Server 2012 R2 / 2016 / 2019
– Minimum 6 Go de RAM
– Minimum 100 Go d'espace disque
– .NET Framework 4.6.1

Architecture recommandée :
– 1 serveur ATA Center (collecte et analyse)
– 1 ATA Gateway installée sur le DC
– Tableau de bord web : https://NomServeurATA
