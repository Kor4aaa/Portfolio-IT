Lab Metasploitable

Kali Linux Metasploit Nmap VMware PHP
🎯 Contexte

Lab de pentest en environnement 100% isolé sous VMware. Cible : la VM Metasploitable 2, intentionnellement vulnérable. Objectif : parcourir les étapes complètes d'une intrusion réelle de façon éthique et documentée.

Ce TP m'a permis d'appliquer une méthodologie d'attaque structurée : reconnaissance → scan → exploitation → post-exploitation, exactement comme lors d'un vrai pentest professionnel.

⚡ Ce que j'ai réalisé
Scan Nmap complet — découverte de tous les ports ouverts et identification des services exposés (FTP, SSH, HTTP, Samba, MySQL...)
Énumération détaillée des services et identification des versions vulnérables
Exploitation via Metasploit Framework — recherche, sélection et lancement d'exploits adaptés
Génération et déploiement de reverse shells pour obtenir un accès distant à la machine cible
Création de web shells PHP pour persistance et exécution de commandes à distance via HTTP
Documentation complète de chaque étape — rapport de pentest structuré
📈 Ce que ça m'a apporté
🔍 Méthodologie

Maîtrise complète du cycle d'un pentest : recon → scan → exploit → post-exploit

🛠️ Metasploit

Utilisation avancée du framework — modules, payloads, sessions Meterpreter

🧩 Logique offensive

Comprendre comment chaîner des vulnérabilités pour progresser dans un système

📝 Documentation

Rédiger un rapport de pentest clair et exploitable — compétence clé en audit
